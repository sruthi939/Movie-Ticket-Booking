# 🎬 Movie Ticket Booking Platform
 
This is a web-based **cinematic movie ticket booking platform** designed to deliver an immersive and premium online cinema experience. Unlike traditional ticket booking systems, It combines **modern UI design, cinematic visual effects, and interactive 3D animations** to transform the entire booking journey into an engaging digital experience.

The platform enables users to **browse the latest and upcoming movies**, **watch trailers**, **explore movie details**, **select theatres and show timings**, **book seats through an interactive theatre layout**, and **securely complete payments**. Users can also **view their booking history**, **download digital tickets**, and **access QR code-based tickets** for seamless theatre entry.

With responsive design, smooth animations, and realistic theatre interactions, this platform provides an intuitive and visually stunning booking experience across desktop, tablet, and mobile devices.

# 🚀 Getting Started

### 🔹 Clone the Repository

```bash
git clone https://github.com/sruthi939/Movie-Ticket-Booking.git

cd Movie-Ticket-Booking
```

---

### 🔹 Install Dependencies

Backend

```bash
cd backend

npm install
```

Frontend

```bash
cd frontend

npm install
```

Admin

```bash
cd admin

npm install
```

---

### 🔹 Environment Variables

Backend `.env`

```env
PORT=5000

MONGODB_URI=

CLERK_SECRET_KEY=

JWT_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```

Frontend `.env`

```env
VITE_API_URL=http://localhost:5000
VITE_CLERK_PUBLISHABLE_KEY=
```

Admin `.env`

```env
VITE_API_URL=http://localhost:5000
VITE_CLERK_PUBLISHABLE_KEY=
```
---

### 🔹 Run the Application

Backend

```bash
cd backend
npm start
```
Frontend

```bash
cd frontend
npm run dev
```

Admin

```bash
cd admin
npm run dev
```
---

The application will be available at:

- Frontend → http://localhost:5173
- Backend → http://localhost:5000
- Admin → http://localhost:5174 *(or your configured port)*
